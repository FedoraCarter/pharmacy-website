# pharmacy-website
website for local pharmacy project 

This project is a website built for a local pharmacy to display products, services, and contact information in a user-friendly and accessible manner. It is designed to improve the pharmacy's online presence and provide easy navigation for customers.

**Features:**

* Clean and responsive design
* Home page with pharmacy overview
* Products page with detailed listings
* Contact page with email and phone information
* Basic navigation menu across pages

**Tech Stack:**

* HTML5
* CSS3
* JavaScript
* Responsive design techniques

**Project Status:**

* Currently in development
* Ongoing improvements to interactivity and design
